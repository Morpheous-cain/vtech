# VisionTech E-Commerce Platform

**Built by Immersicloud Consulting** · March 2026  
Full-stack Next.js e-commerce platform for Nairobi-based electronics retailer.



## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database & Auth**: Supabase (Postgres + JWT)
- **Styling**: Tailwind CSS + Radix UI
- **Payments**: M-Pesa Daraja API + Flutterwave
- **Images**: Cloudinary CDN
- **Deployment**: Vercel


## Key Features
- 🛍️ Full product catalogue with search & filtering
- 🛒 Persistent cart (guest + logged-in)
- 📱 M-Pesa STK Push payment flow
- 💳 Flutterwave card payments
- 📦 Order tracking (Packed → Dispatched → Delivered)
- 🏪 Admin dashboard (products, orders, inventory, analytics)
- 🔒 RLS-protected Supabase database
